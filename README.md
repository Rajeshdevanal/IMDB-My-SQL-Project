**IMDb Movie Database (SQL Project)**
**📌 Overview**

This project is a relational movie database inspired by IMDb. It stores and analyzes movie data such as titles, genres, release years, ratings, and votes using SQL.

**🗂️ Features**

Normalized database design

Primary & foreign key relationships

Complex SQL queries

Genre-wise and rating-based analysis

🛠️ Technologies Used

SQL / MySQL

**📊 Key Analysis**

Top-rated movies

Genre-wise average ratings

Most voted movies

**🎯 Learning Outcomes**

Database normalization

Joins, subqueries, aggregations

Real-world SQL problem solving
